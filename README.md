# Python Lab Programs

This repository contains small Python practice programs for basic programming and lab/tutorial exercises. The scripts focus on beginner-friendly topics such as input/output, conditionals, loops, functions, recursion, lists, and tuples.

## Topics Covered

- Basic input and output
- `if` / `elif` / `else` conditions
- Factorial and prime number checks
- String operations
- List sorting and searching
- Running sum and average calculation
- Recursion
- Tuple-based exercises

## Requirements

- Python 3.x

## How to Run

Run any script from the repository folder:

```bash
python 1Basic_input_output.py
```

If your system uses `python3`, run:

```bash
python3 1Basic_input_output.py
```

Most files take input from the keyboard when executed.


## Notes

- The repository appears to be a collection of lab/tutorial exercises rather than a single application.
- Some scripts are complete programs, while a few only define functions and do not yet include user interaction or output.
